# 📸 LensCraft Studio — Premium Photography Website

A world-class, production-ready photography studio website built with **HTML5, CSS3, JavaScript (ES6), and Bootstrap 5**. Designed with a luxury, cinematic, dark-themed aesthetic featuring gold accents, glassmorphism, and smooth animations.

## ✨ Features

### Design & UI
- 🎨 Dark luxury theme with gold (#D4AF37) accent color
- 💎 Glassmorphism cards with backdrop blur effects
- 🌟 Smooth scroll-triggered reveal animations
- 📱 Fully responsive across all devices (desktop, tablet, mobile)
- 🔤 Premium typography (Playfair Display + Poppins)
- 🎭 Parallax background effects
- 🖼️ Image zoom hover effects
- ✨ Micro-animations and hover transitions

### Functionality
- ⌨️ Typing effect on hero section
- 📊 Animated counters with IntersectionObserver
- 🖼️ Gallery filtering by category (7 categories, 24+ images)
- 🔍 Lightbox with keyboard navigation (prev/next/escape)
- 📝 Contact form with real-time JavaScript validation
- 📧 Newsletter subscription with success notifications
- 🔝 Back-to-top button with smooth scroll
- 🍔 Sticky navbar with transparency-to-solid transition
- 🎯 Active navigation highlighting
- ⏳ Loading screen animation

### SEO & Accessibility
- ✅ Semantic HTML5 structure
- ✅ Meta titles & descriptions on every page
- ✅ Open Graph & Twitter Card meta tags
- ✅ Proper heading hierarchy (single H1 per page)
- ✅ Alt attributes on all images
- ✅ ARIA labels on interactive elements
- ✅ Keyboard-accessible navigation
- ✅ Fast-loading with CDN assets
